# ReplaceFont
## Description
Good-by bitmap font. This script replace font in TextStyle.plist for ja.

!["sample"](./itunesfont.png)

## Usage
   1. Download this script.
   2. cd folder that you saved this script.
   3. ./ReplaceFont.ps1 arial
   4. Open iTunes for Windows.
